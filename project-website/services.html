Services offered by Avis software app

1. Account and Security Service:

The app requires the user to first login or sign up first in order to bring in the aspect of security in terms of confidentiality,Integrity, and Availability. this ensures that the user`s data is safe and accessible by the rightful owner. This service was implemented using the a Firebase Auth Service of the Firebase API.

2. Database Service:

This is awesome service which stores the data of the user(reminders). lt ensures that the data is available when the user re-opens or re-runs the app again. ie no loss of data. This service was implemented using the Hive database.

3. Location Service:

This service helps to track the postion of the user, and showing the user`s current position. This was dominant in calculating the remainder distance which the user is remaining with, to reach the selected destination It was implemented using the geolocator and geocoding package dependencies, and the Open Street Map API for displaying the map.

4. Notification Service:

The nofitication alerts and alarms are the main key driving this app, the ability to notify the user to reminnd him/her is an important aspect ie scheduled and location-based notifications. This was implemented the flutter local notifications dependency package.
